# WanderLu Project
WanderLust is a web application built using Node.js, Express, MongoDB, and EJS.
This project allows users to create, view, edit, and delete travel listings and also add reviews to listings.

# Project Features
view all listings
Add a new listing
Edit a listing
Delete a listing
Add reviews to a listing
Server-side validation using Joi
Error handling with custom error pages

# Technologies Used
Node.js
Express.js
MongoDB
Mongoose
EJS & EJS-Mate
Joi Validation
Method Override
